## Run Locally


Clone the project

```bash
  git clone https://github.com/amilali/TalkifyDocs.git
```

Go to the project directory

```bash
  cd talkifydocs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
